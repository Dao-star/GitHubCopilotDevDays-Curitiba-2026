If you encounter problems during the setup or execution, refer to the following solutions:

- Permission Denied: Right-click the PowerShell icon and select Run as Administrator.
- Connection Failed: Check your firewall settings. The software needs access to github.com and connected AI model services.
- Version Mismatch: Visit the download page again to ensure you are using the latest version of the software.
- Login Timeout: Close the application and restart the initial setup process if the browser authentication page does not appear.
